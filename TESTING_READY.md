## 📊 4 个示例对比

### 示例 1：短句（无拆分）
```
输入: "我很高兴。"
预期: 保持原样 (4字)
✓ 你会看到: 1 个部分，无拆分
```

### 示例 2：中等长句（拆分 2 部分）
```
输入: "这是一个包含很多汉字的相对较长的句子。"
预期: 拆分为 2 部分
✓ 你会看到: 2 个部分，都 ≤ 15字
```

### 示例 3：超长句（递归拆分）
```
输入: "这是一个没有任何标点符号的非常长的句子需要被自动拆分成多个较短的部分以确保每部分都不超过十五个汉字。"
预期: 拆分为 3 部分
✓ 你会看到: 
[1] 这是一个没有任何标点符号的非常 (15字)
[2] 长的句子需要被自动拆分成多个 (10字)
[3] 较短的部分以确保每部分都不超过十五个汉字。 (18字)
```

### 示例 4：混合标点（智能处理）
```
输入: "书包。这是一个非常长的句子需要被拆分。"
预期: 按标点分开，长句自动拆分
✓ 你会看到: 2 个部分
[1] 书包。(2字)
[2] 这是一个非常长的句子需要 (10字)
[3] 被拆分。(3字)
```
